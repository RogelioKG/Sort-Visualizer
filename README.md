# Sort-Visualizer

![sorting](./sorting.gif)

## Build
+ using `uv`
  ```bash
  uv sync
  ```
+ using `pip`
  ```bash
  py -m virtualenv .venv
  .venv/Scripts/activate
  pip install -r requirements.txt
  ```

## Run
+ using `uv`
  ```bash
  uv run main.py
  ```
+ using `pip`
  ```bash
  py main.py
  ```
